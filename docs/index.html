Get started in 2 minutes

Download & unzip

Grab Sotto-windows-x64.zip from Releases and unzip it.

Run Sotto.exe

SmartScreen may warn (unsigned) → More info → Run anyway.

Download the model

Settings → Model → Download (one-time, ~300 MB).

Speak

Hold Ctrl + Win, talk, release. Text appears at your cursor. 🎉

FAQ

Is my audio uploaded anywhere?

No. Transcription runs entirely on your CPU. The only network request is the one-time model download you start yourself.

Does it need a GPU?

No — it's tuned for CPU and runs on a normal laptop. You can optionally use an Intel iGPU.

Windows says "unknown publisher."

The app is unsigned (certificates cost money; Sotto is free). Click More info → Run anyway, or build it from source.

Which languages are supported?

English and Hindi/Hinglish out of the box, with one-click switching. Set “Auto — all languages” for ~99 Whisper languages.
